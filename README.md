# Приглашение для Томирис

Сайт-приглашение на 14 февраля: тёмная элегантная тема, мини-игры для разблокировки событий плана дня.

## Запуск

```bash
cd invitation
npm install
npm run dev
```

Открой в браузере адрес, который покажет Vite (обычно http://localhost:5173).

## Сборка

```bash
npm run build
```

Готовые файлы будут в папке `dist/`.

## Что внутри

- **React 18 + Vite** — быстрый старт и сборка
- **Framer Motion** — плавные анимации появления и интро
- **Тёмная палитра** — тёмный фон, розовые и золотые акценты, шрифты Cormorant Garamond и Outfit
- **Три мини-игры** для разблокировки событий: сердечки, арканоид, Block Blast
- Тот же контент, что и в референсе: гончарная, ужин «Алаша», глэмпинг, чеклист, календарь, сообщение и подпись

Все тексты и ссылки заданы в `src/data.js` — их можно править под себя.
